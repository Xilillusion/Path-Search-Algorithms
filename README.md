# Path Search Algorithms AIO
An all-in-one path search algorithms for grid-like maze, including A*, Bi-HS, MMA*, BFS, and Bi-BFS
## Files
`AStar.py`
- `A_star`: A*
- `Bi_HS`: Bidirectional Heuristic Search (Bidirectional A*)
- `MM`: MMA* (Meet-in-the-Middle A*)

`BFS.py`
- `BFS`: Breadth First Search
- `Bi_BFS`: Bidirectional BFS

`maze_generator.py`
- `dfs_maze`
- `kruskal_maze`
- `prim_maze`
- `aldous_broder_maze`
